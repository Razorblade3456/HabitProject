# BugBite

BugBite is an Expo + React Native (TypeScript) prototype for a habit and todo tracker with a bug-smashing twist.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the Expo development server:
   ```bash
   npm run start
   ```

3. Run on a simulator/device:
   - Press `i` for iOS
   - Press `a` for Android
   - Press `w` for web

## Project Structure

- `App.tsx` - Navigation setup
- `src/screens` - Tab screens
- `src/components` - Shared components (empty)
- `src/state` - State management (empty)
- `src/theme.ts` - Dark neon theme tokens
